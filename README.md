# ATTiny84A + nRF24L01 Transmit Data Example

Example of transmitting data from an ATTiny84A with an nRF24L01 radio module  

## Hardware
- Microcontroller -> Atmel ATTiny84a
- Radio -> nRF24L01 module
- Programmer -> Atmel AVRISP MkII

## Software
- VS Code + Atmel C Project Generator extension
- avr-gcc
